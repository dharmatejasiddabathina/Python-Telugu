# Python Variables and Data Types Explanation

This guide is intended for beginners learning Python. It covers the core concepts of **variables** and key **data types**: `int`, `float`, `str`, and `bool`. The purpose is to make understanding these concepts straightforward and accessible for everyone.

***

## What Are Variables?

- Variables are named containers used to store data values.
- In Python, you assign a value to a variable using the `=` operator.
- Python assigns data types automatically, depending on the value you assign.
- Variable names must:
  - Start with a letter or underscore (`_`)
  - Contain only letters, numbers, or underscores
  - Not be a Python keyword
  - Be case-sensitive (`age` and `Age` are different)

**Example:**

```python
age = 25           # int
name = "Alice"     # string
is_student = True  # boolean
```

***

## Integer (`int`)

- Represents whole numbers, positive or negative.
- Python supports integers of arbitrary size.

**Example:**

```python
score = 100
print(score)          # Outputs: 100
print(type(score))    # Outputs: lassss 'int'>
```

***

## Floating Point (`float`)

- Represents real numbers with a decimal point.
- Used for precise values like measurements or prices.

**Example:**

```python
temperature = 36.6
print(temperature)        # Outputs: 36.6
print(type(temperature))  # Outputs: <class 'float'>
```

***

## String (`str`)

- Sequence of characters such as letters, digits, symbols.
- Defined by wrapping text with `' '` or `" "`.

**Example:**

```python
greeting = "Hello, World!"
print(greeting)        # Outputs: Hello, World!
print(type(greeting))  # Outputs: <class 'str'>
```

**Concatenate strings:**

```python
first_name = "John"
last_name = "Doe"
full_name = first_name + " " + last_name
print(full_name)     # Outputs: John Doe
```

***

## Boolean (`bool`)

- Represents true or false conditions.
- Used in decision-making and control flow.

**Example:**

```python
is_raining = False
print(is_raining)        # Outputs: False
print(type(is_raining))  # Outputs: <class 'bool'>
```

***

## Checking Data Types

Use `type()` to find out a variable's data type:

```python
x = 42
print(type(x))           # <class 'int'>

y = 3.14
print(type(y))           # <ass 'float't'>

z = "Python"
print(type(z))           # <class 'str'>

flag = True
print(type(flag))        # lassss 'bool'>
```

***

## Type Conversion (Casting)

Convert data from one type to another:

```python
num_str = "123"
num_int = int(num_str)

print(num_int)          # Outputs: 123
print(type(num_int))    # lass 'intnt'>

pi = 3.14159
pi_str = str(pi)

print(pi_str)           # Outputs: '3.14159'
print(type(pi_str))     # lass ' 'str'>
```

***

## Naming Rules & Best Practices

- Use descriptive names: `student_age`, `total_amount`.
- Use snake_case (lowercase with underscores).
- Names must not start with a number.
- Avoid Python reserved keywords (`if`, `else`, `class`).
- Variable names are case-sensitive.

***

## Full Example Program

```python
# Python Variables and Data Types

# Integer
age = 30
print("Age:", age, "| Type:", type(age))

# Float
height = 5.9
print("Height:", height, "| Type:", type(height))

# String
name = "Jane Smith"
print("Name:", name, "| Type:", type(name))

# Boolean
is_student = True
print("Is student:", is_student, "| Type:", type(is_student))

# Convert string to integer
score_str = "88"
score_int = int(score_str)
print("Score as string:", score_str, "| Type:", type(score_str))
print("Score as integer:", score_int, "| Type:", type(score_int))
```

***

## Summary

- Variables are used to store data.
- Data types include integers, floats, strings, and booleans.
- You can check the data type with `type()`.
- Convert types using functions like `int()`, `str()`, and `float()`.
- Follow naming conventions for clearer, maintainable code.
