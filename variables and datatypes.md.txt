```markdown
# Detailed Explanation of Python Variables and Data Types

This guide is designed for beginners learning Python variables and key data types: `int`, `float`, `string`, and `bool`. It is perfect for classroom teaching or YouTube tutorials.

---

## What Are Variables?

- Variables are named containers used to store data values in a program.
- In Python, assign a value to a variable using the equals sign (`=`).
- Python is dynamically typed; it infers the data type automatically.
- Variable names must start with a letter or underscore, can contain letters, numbers, or underscores, cannot be Python keywords, and are case-sensitive.

**Example:**

```
age = 25           # int
name = "Alice"     # string
is_student = True  # boolean
```

---

## Integer (`int`)

- Whole numbers without decimals (positive or negative).
- Python supports arbitrarily large integers.

**Example:**

```
score = 100
print(score)          # 100
print(type(score))    # <class 'int'>
```

---

## Floating Point (`float`)

- Numbers with decimal points.
- Used for precise measurements, prices, etc.

**Example:**

```
temperature = 36.6
print(temperature)        # 36.6
print(type(temperature))  # lass ' 'float'>
```

---

## String (`str`)

- Sequence of characters enclosed in single `' '` or double `" "` quotes.
- Supports concatenation and formatting.

**Example:**

```
greeting = "Hello, World!"
print(greeting)           # Hello, World!
print(type(greeting))     # lass 'strtr'>
```

Concatenation example:

```
first_name = "John"
last_name = "Doe"
full_name = first_name + " " + last_name
print(full_name)          # John Doe
```

---

## Boolean (`bool`)

- Logical values: True or False.
- Used in conditions and comparisons.

**Example:**

```
is_raining = False
print(is_raining)         # False
print(type(is_raining))   # lass 'boolol'>
```

---

## Checking Variable Types

Use `type()` to check a variable’s data type:

```
x = 42
print(type(x))           # lass 'intnt'>

y = 3.14
print(type(y))           # lass 'floatat'>

z = "Python"
print(type(z))           # lass 'strtr'>

flag = True
print(type(flag))        # lass 'boolol'>
```

---

## Type Conversion (Casting)

Convert variables from one type to another with:

- `int()`, `float()`, `str()`, `bool()`.

**Example:**

```
num_str = "123"
num_int = int(num_str)

print(num_int)           # 123
print(type(num_int))     # lass 'intnt'>

pi = 3.14159
pi_str = str(pi)

print(pi_str)            # '3.14159'
print(type(pi_str))      # lass 'strtr'>
```

---

## Naming Rules and Best Practices

- Use descriptive, meaningful names.
- Use `snake_case` for multi-word names, e.g., `student_count`.
- Names must start with letters or underscores, not numbers.
- Avoid Python keywords (`class`, `for`, `if`).
- Variable names are case-sensitive (`age` ≠ `Age`).

```

```
## Summary for Students

- Variables store and label data values.
- `int` is for whole numbers, `float` for decimals.
- `str` holds text, `bool` stores True/False values.
- Use `type()` to check data types.
- Convert types with casting functions when needed.
- Follow naming best practices for clean code.
```
